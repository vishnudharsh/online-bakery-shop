# Online Bakery Shop 🍰

A full-stack e-commerce web application for ordering bakery products online.

## Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Styling: Bootstrap / CSS

## Features
- User registration & login
- Browse bakery products
- Add to cart
- Order management
- Admin product management

## Installation
git clone https://github.com/vishnudharsh/online-bakery-shop.git
cd online-bakery-shop

# Install server
cd server
npm install

# Install client
cd ../client
npm install

## Run the project
cd server
npm start

cd ../client
npm start
